Objetivo do Projeto

O principal objetivo deste projeto é desenvolver uma aplicação web robusta, modular e responsiva utilizando o ecossistema Angular e TypeScript. O sistema conecta-se à API pública Hipo Labs Universities API para mapear e exibir informações detalhadas sobre universidades de todo o mundo.

Esta aplicação foi desenvolvida como parte da avaliação acadêmica, exercitando conceitos cruciais como a manipulação de dados com o HttpClient do Angular, persistência com o Local Storage, injeção de dependências em serviços, controle de fluxos através de rotas dinâmicas, data-binding e diretivas.

Conceitos Obrigatórios Aplicados

Angular Core: Componentes customizados, Serviços Compartilhados, Injeção de Dependências e Interfaces TypeScript tipadas.
Rotas: Navegação dinâmica e parametrizada via RouterModule.
Diretivas: Estruturação condicional com *ngIf e loops com *ngFor.
Data Binding: Interpolação, Binding de propriedades ([href], [ngClass]), Binding de eventos ((click)) e Two-Way Binding ([(ngModel)]).
